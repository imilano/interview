    15. 二进制中 1 的个数
      • Description
      • Solutions
        • Bit Manipulation

    Description #

    输入一个整数 n ，输出该数32位二进制表示中1的个数。其中负数用补码表示。

    Solutions #

    Bit Manipulation #

    这里需要注意的是，因为负数用补码表示，而负数补码右移会导致高位补 1，所以对负数不能直接进行右移。

    func NumberOf1( n int ) int {
    // write code here
    var count int
    t := 0x80000000
    for n != 0 {
        count += (n & t) >> 31
        n <<= 1
    }
    
    return count
}
