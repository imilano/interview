### Bit Manipulation

使用位与操作表示两数相加的进位（需要左移一位）， 使用异或操作表示两数相加的和，这样就可以递归计算，递归终止条件是进位为 0.
```go
func Add( num1 int ,  num2 int ) int {
    // write code here
    if num1 == 0 {
        return num2
    }
    
    carry := (num1 & num2) << 1
    sum := num1 ^ num2
    return Add(carry, sum)
}
````
