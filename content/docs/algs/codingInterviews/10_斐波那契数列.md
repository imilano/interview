
## Description

> 输入一个正整数 n ，请你输出斐波那契数列的第 n 项


## Solutions

简单题，简单的动态规划。
```go
func Fibonacci( n int ) int {
    // write code here
    if n <= 2 {
        return 1
    }
    
    NMinusOne, NMinusTwo := 1, 1
    for i := 3; i <= n; i++ {
        N := NMinusOne + NMinusTwo
        NMinusTwo = NMinusOne
        NMinusOne = N
    }
    
    return NMinusOne
}
```
